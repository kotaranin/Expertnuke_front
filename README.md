# ExpertNuke - Frontend

React + Tailwind CSS frontend for the ExpertNuke expert system. Guides the user step by step through a series of questions about the current state of a nuclear reactor, then displays the recommendations returned by the backend.

## Tech Stack

- React
- Tailwind CSS

## How to Run

1. Install dependencies:
```bash
   npm install
```
2. Start the dev server:
```bash
   npm run dev
```
3. Open the URL `http://localhost:5173`.

**Note:** the backend must be running on `http://localhost:8081` for the analysis start (see backend README).
